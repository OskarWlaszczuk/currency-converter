# Currency-converter project

## Description: 
Here is my personal currency-conventer project that will immediately help you find out how much Polish zlotys are worth in other currencies. It is simple to work with.

## Languages & Conventions:

- *HTML*
- *CSS*
    - *BEM* convention
    - *Flex-Box*
    -  CSS Backgrounds and Borders Module Level 3
    - Media Queries
        - *Mobile-first*
- *Java Script*
- Open Graph Protocol

## How to use it?
1. First of all, you have to write numbers between 0.1 and 999999 in the form labeled *"Kwota w PLN"* (eng. *"Amount in PLN"*).

2. Then, you have to choose the currency that you are interested in, for example:
- EUR
- USD
- or CAD

You can do this by clicking the circle next to the currency name. 

3. In the last step you need to click orange button named *"Przelicz"* (eng. *"Convert"*). The converted amount will be displayed next to *"Wynik"* (eng.*"Result"*).
